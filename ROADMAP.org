* CLatter Development Roadmap

** Completed Features

*** v0.1 - Initial Release
- [X] Basic IRC protocol implementation (RFC 1459/2812)
- [X] TLS/SSL support
- [X] Multi-buffer interface (server, channels, queries)
- [X] NickServ auto-identify
- [X] Channel autojoin
- [X] Input history (Up/Down arrows)
- [X] Scrollback with Page Up/Down
- [X] Basic commands: /join, /part, /msg, /me, /nick, /quit, /query, /raw
- [X] Service shortcuts: /ns, /cs
- [X] Nick colorization (hash-based consistent colors)
- [X] Nick highlighting (visual indicator when mentioned)
- [X] Tab completion for nicks
- [X] Autojoin management (/autojoin add/remove)
- [X] /q shortcut for /quit

*** v0.2 - Polish & Usability
- [X] CTCP support (VERSION, TIME, PING)
- [X] /whois command
- [X] /topic command
- [X] /kick and /ban commands
- [X] /mode command
- [X] Timestamps on messages (configurable format)
- [X] /ctcp command for sending CTCP requests

*** v0.2.1 - Resilience
- [X] Auto-reconnect with exponential backoff
- [X] Connection health monitoring (idle detection, ping/pong)
- [X] Graceful handling of connection drops

** Planned Features

*** v0.3 - Logging & Search
- [X] Message logging to disk
- [X] Log search functionality
- [X] Log viewer/browser (/log command)
- [X] Export logs to various formats (text, JSON, HTML)

*** v0.4 - Multi-Network
- [ ] Multiple simultaneous network connections
- [ ] Network switching commands
- [ ] Per-network buffer organization
- [ ] /server command to add/connect networks

*** v1.0 - Daemon Mode (Major)
- [ ] Separate IRC backend daemon process
- [ ] Systemd service integration
- [ ] Persistent connections (survive TUI restarts)
- [ ] Message buffering while TUI disconnected
- [ ] Unix socket IPC between daemon and TUI
- [ ] Attach/detach model (like tmux/screen)
- [ ] SQLite message storage
- [ ] Web interface option (stretch goal)

** Ideas / Wishlist

- [ ] Plugin/scripting system (maybe via Lisp eval)
- [ ] Theming support (color schemes)
- [ ] Mouse support for buffer switching
- [X] Split-pane view for multiple channels
- [ ] DCC file transfer support
- [X] IRCv3 capabilities (SASL PLAIN authentication)
- [ ] Matrix bridge support
- [ ] Spell checking integration
- [ ] Channel modes display
- [ ] User list panel (toggleable)
- [ ] URL detection and highlighting
- [ ] Desktop notifications (libnotify)
